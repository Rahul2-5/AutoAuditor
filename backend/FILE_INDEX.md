## 🚀 Quick Start (Choose Your Path)

### Path 1: Postman Testing (RECOMMENDED)
1. Open Postman
2. **File → Import** → [`Postman_Auditor_Collection.json`](Postman_Auditor_Collection.json)
3. Click any test in collection
4. Click **Send**
5. View JSON response

**Time:** 2 minutes  
**Documentation:** [`POSTMAN_TESTING_SUMMARY.md`](POSTMAN_TESTING_SUMMARY.md)

### Path 2: Python Testing
1. Terminal: `cd backend`
2. Run: `python test_auditor.py`
3. View output: [`audit_report_output.json`](audit_report_output.json)

**Time:** 30 seconds  
**See:** [`AUDITOR_README.md`](AUDITOR_README.md)

### Path 3: Manual API Testing
1. Use cURL or any HTTP client
2. POST to `http://localhost:8000/api/analyze`
3. Send JSON request body
4. Receive JSON response

**Time:** 5 minutes  
**Reference:** [`AUDITOR_API_DOCS.md`](AUDITOR_API_DOCS.md)
